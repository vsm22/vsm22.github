Class: Widget

(abstract) Widget(container, oopt)

Abstract base class representing an SVG widget that can be placed inside a DOM container. Classes implementing this abstract class must implement the following: 1) _initOptions(o) 2) _initStateConstraints() 3) _initState() 4) _initSvgEls() 5) _initHandlers() 6) getVal() 7) _update()
