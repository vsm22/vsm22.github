(abstract) setInternalState(newStateopt) → {boolean}

Set the current state and redraw. If no new state argument is provided, will reassign old state, taking into account the stateConstraints. As opposed to setState(), setInternalState() does not trigger observer notification. Will use Widget.stateConstraints to constrain each state value to each constraints min, max, or enum
